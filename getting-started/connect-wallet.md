# Connect Wallet

Arbital uses wallet-based authentication — no email or password required.

## Supported Wallets

### EVM (Ethereum, Base, Arbitrum)
- MetaMask
- Rabby
- WalletConnect
- Coinbase Wallet

### Solana
- Phantom
- Solflare
- Backpack

## Connection Steps

1. Click **Connect Wallet** on the homepage
2. Select your wallet type
3. Approve the signature request
4. You're connected!

{% hint style="warning" %}
Arbital only requests a signature to verify ownership. We never request transaction approvals during login.
{% endhint %}

## Switching Wallets

1. Click your wallet address (top right)
2. Select **Disconnect**
3. Connect with a different wallet

## Multiple Wallets

You can link multiple wallets to one Arbital account:

1. Go to **Settings → Wallets**
2. Click **Link Additional Wallet**
3. Connect and sign with the new wallet

## Troubleshooting

**Wallet not connecting?**
- Refresh the page
- Check if wallet extension is unlocked
- Try a different browser
- Clear browser cache

**Wrong network?**
- Arbital auto-detects networks
- Ensure your wallet is on a supported chain
