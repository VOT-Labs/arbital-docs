# Quick Start

Get your first Arbital bot running in under 10 minutes.

## Prerequisites

- A Web3 wallet (MetaMask, Rabby, Phantom, etc.)
- Funds on a supported exchange
- API keys from your chosen exchange

## Step 1: Connect Your Wallet

1. Visit [arbital.xyz](https://arbital.xyz)
2. Click **Connect Wallet**
3. Select your wallet provider
4. Approve the connection

## Step 2: Add Exchange Credentials

1. Navigate to **Settings → Exchanges**
2. Select your exchange (e.g., Lighter, Pacifica)
3. Enter your API Key and Secret
4. Click **Verify Connection**

{% hint style="info" %}
Your API keys are encrypted and stored securely. We recommend using read/trade-only keys without withdrawal permissions.
{% endhint %}

## Step 3: Create a Bot

1. Go to **Dashboard → Create Bot**
2. Select exchange and market (e.g., BTC-PERP)
3. Configure parameters:
   - **Spread**: 0.1% (default)
   - **Order Size**: Based on your balance
   - **Rebalance Threshold**: 5%
4. Click **Deploy Bot**

## Step 4: Monitor

Your bot is now live! Monitor performance from the dashboard:

- Real-time P&L
- Position sizes
- Order history
- Funding earned
