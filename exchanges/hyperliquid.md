# Hyperliquid

Hyperliquid is a high-performance L1 built specifically for trading.

## Overview

- **Chain**: Hyperliquid L1
- **Settlement**: USDC
- **Leverage**: Up to 50x
- **Status**: Fully supported

## Setup Guide

### 1. Onboard to Hyperliquid

1. Visit [hyperliquid.xyz](https://hyperliquid.xyz)
2. Connect EVM wallet
3. Bridge USDC to Hyperliquid
4. Deposit to trading account

### 2. Create API Wallet

Hyperliquid uses a unique API wallet system:

1. Go to **More → API**
2. Click **Create API Wallet**
3. Save the generated private key securely
4. Fund the API wallet if required

### 3. Add to Arbital

1. In Arbital: **Settings → Exchanges**
2. Select **Hyperliquid**
3. Enter your API wallet private key
4. Verify connection

{% hint style="warning" %}
The API wallet private key is sensitive. Store it securely and never share it.
{% endhint %}
