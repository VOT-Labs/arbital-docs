## [1.2.0] - 2026-02-02

### Added
- Dark mode support for documentation
- Risk disclosure page and compliance banners
- Visual architecture diagrams
- Flow diagrams for credential binding, strategies, and delta-neutral concepts
- "Last updated" timestamps on all documentation pages

### Changed
- Quick Links section now uses card layout for better UX
- Improved Getting Started flow with clearer steps
- Standardized all icons to Font Awesome (GitBook-native) for consistent dark/light mode rendering

---

## [1.1.0] - 2026-01-15

### Added
- Grid Market Maker strategy
- Extended (Hyperliquid) exchange support
- Bot lifecycle state management
- Inventory management with configurable skew factors
- Advanced execution modes (Aggressive, Normal, Passive)

### Changed
- Improved credential binding flow
- Enhanced dashboard analytics

### Fixed
- WebSocket reconnection handling
- Balance sync delays on Pacifica

---

## [1.0.0] - 2025-12-01

### Added
- Initial platform launch (Beta)
- TWAP Market Maker strategy
- Lighter exchange integration
- Pacifica exchange integration
- Non-custodial credential binding
- Real-time bot monitoring dashboard
- Balance and volume charts
- Events table with order tracking

### Security
- Wallet-based authentication (no passwords)
- EIP-712 typed data signing for EVM
- Solana message signing
- Trading-only permissions (no withdrawal access)
